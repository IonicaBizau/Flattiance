# Flattiance [![Support this project][donate-now]][paypal-donations]

A semi-flat fork of the Ubuntu Ambiance theme.

## Installation

Copy the `Flattiance` directory into `/usr/share/themes/` and activate it using `unity-tweak-tool`.
